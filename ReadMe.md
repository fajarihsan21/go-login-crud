## Installation Steps

1. Clone the repository

```bash
git clone https://github.com/fajarihsan21/go-login-crud.git
```

2. Install dependencies

```bash
go get -u ./...
# or
go mod tidy
```

3. Run the app

```bash
go run *.go
# or
go run main.go
```

You are all set!
